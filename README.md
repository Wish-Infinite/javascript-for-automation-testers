Welcome to the **JavaScript for Automation Testers** repository 👨‍💻✨

No unnecessary theory. No confusion.

# 🎯 What You’ll Learn

This repository covers all essential JavaScript topics required for automation:

- ✅ Variables (`let`, `const`)  
- ✅ Data Types (String, Number, Boolean, Array, Object)  
- ✅ Functions & Arrow Functions  
- ✅ Operators (Arithmetic, Comparison, Logical)  
- ✅ Conditions (`if / else`)  
- ✅ Loops (`for`, `for...of`)  
- ✅ Async & Await (🔥 most important)  
- ✅ Classes & Methods  
- ✅ Objects & Method Calling  
- ✅ Constructors  
- ✅ Import / Export (Modules)
